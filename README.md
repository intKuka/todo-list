## Usings

- NodeJS v20.10.0
- PostgreSQL
- Prisma ORM

## Installation

```bash
$ npm install
```

## Setting & Configuration

Need to create and configure .env file

_Possible enviroments:_

```bash
# development file
.development.env

# production file
.production.env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
